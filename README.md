> Telegram Nmap Scan Bot

## Requirements

* [Nmap](https://nmap.org/)
* Python
* Pyrogram
* speedtest-cli

## Tested On

* Linux

### Self-Hosting

```sh
# Install Nmap
$ sudo apt instll nmap -y
# Install Git & Python3
$ sudo apt install git python3 python3-pip -y
# Clone Repo
$ git clone https://github.com/Dilum125/Nmap-Scan-Bot
$ cd Nmap-Scan-Bot
# Install Requirements 
$ pip3 install -r requirements.txt
# Start bot 
$ python3 bot.py
```
